LibreLock ‚Äî My First Secure Password Manager

LibreLock is a lightweight, secure, and open-source password manager built for the extra paranoid. It stores your credentials encrypted with Argon2id + Fernet, and keeps sensitive data safe even if someone gets access to the database file.

Developed with Python, SQLite, and CustomTkinter for the UI, it features password storage, retrieval, secure copy-to-clipboard, editing, and deletion, plus notes and URL management for every service. All encryption keys are derived from the master password; without it, the data is practically impossible to decrypt.

Full disclosure: this is my first proper program, coded fueled by lots of coffee, too many cigarettes, a lot of googling, and asking ChatGPT. I have zero design skills, so the UI is as functional and clean as ChatGPT could make it. The app supports adding notes, URLs, and safe copy-to-clipboard functionality.

Technical Details

  • Passwords, usernames, emails, URLs, and notes are encrypted using Fernet (AES 128 in CBC mode with HMAC) with keys derived from Argon2id.
  • Each password entry has its own encryption salt; master password changes re-encrypt all stored data safely.
  • Search tags for services are hashed with HMAC-SHA256, so even service names are not stored in plaintext.
  • Safe copy-to-clipboard clears itself after 30 seconds to reduce leakage risk.
  • No sensitive info is logged; logs mostly track actions like "user added a password" without revealing actual data.
